# Wordpedia

Wordpedia is a minimal dictionary web application that allows users to search and discover the meaning of English words quickly and easily.

This project focuses on simplicity, speed, and a clean user interface to make learning and understanding words more enjoyable.

---

## Features

* Search for word definitions instantly
* View phonetic spelling
* Listen to word pronunciation audio
* See word meanings and examples
* Minimal and modern user interface

---

## Tech Stack

This project was built using:

* **React**
* **Tailwind CSS**
* **Dictionary API**

---

## Installation

Clone the repository:

```
git clone https://github.com/your-username/wordpedia.git
```

Navigate to the project folder:

```
cd wordpedia
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

---

## API Used

This project uses the free dictionary API:
